# Human Resources Analytics Dashboard

## Overview

This project presents an interactive **Human Resources Analytics Dashboard** built in Tableau to analyze workforce composition, hiring and termination trends, employee demographics, compensation, performance, and geographic distribution.

The dashboard is designed to help HR teams and business leaders monitor workforce trends, identify patterns across employee groups, and support data-driven workforce planning.

The project includes two primary dashboards:

- **HR Summary Dashboard**
- **Employee Details Dashboard**

---

## Project Objectives

The dashboard was developed to:

- Monitor total hired, terminated, and active employees
- Analyze workforce trends over time
- Compare hiring and termination patterns across departments
- Examine employee demographics by age, gender, and education level
- Evaluate salary and performance relationships
- Explore workforce distribution across states, cities, headquarters, and branches
- Provide detailed employee-level records for further analysis

---

## Dashboard Features

### HR Summary Dashboard

The summary dashboard provides a high-level view of workforce trends, including:

- Total employees hired
- Total employees terminated
- Total active employees
- Hiring trends over time
- Termination trends over time
- Department-level workforce distribution
- Job title analysis
- Gender distribution
- Education-level analysis
- Age-group segmentation
- Salary analysis
- Performance rating analysis
- Geographic workforce distribution

---

### Employee Details Dashboard

The employee detail view allows users to explore individual employee records and filter the workforce by multiple dimensions.

Key fields include:

- Employee information
- Department
- Job title
- Gender
- Age
- Education level
- Salary
- Performance rating
- Employment status
- Hire date
- Termination date
- State and city
- Headquarters vs. branch location

---

## Calculated Fields

Custom Tableau calculated fields were created to support workforce analysis, including:

- Total Hired
- Total Terminated
- Total Active Employees
- Employee Age
- Age Groups
- Employment Status
- Length of Hire / Tenure
- Headquarters vs. Branch Classification
- Percentage of Total Hired
- Percentage of Total Terminated

These calculations help transform raw employee records into business-relevant HR metrics.
